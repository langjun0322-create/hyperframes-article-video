# Clear Code Theme

Clear Code is a warm, clear, human-centered editorial theme for AI tools, workflows, release notes, tutorials, and practical explainers.

It is inspired by clean product documentation and utility-focused visual systems. It does not copy any external brand identity. The theme uses local fonts, CSS backgrounds, and registered SVG line icons so generated videos stay deterministic.

## Style

- Warm off-white canvas with a thin rounded frame.
- Large high-contrast serif titles.
- Terracotta orange accents.
- Fine line icons, dotted fields, clean linear canvases, and calm spacing.
- Light cards only where information needs grouping.

## Asset Rules

- Do not generate per-article backgrounds.
- Use `theme.json.background_presets` for CSS-only deterministic canvases.
- Use only registered SVG icons from `asset_registry.icons`.
- Keep cards, diagrams, buttons, badges, and nav bars as HTML/CSS inside the safe area.
- Do not use the reference brand's names, logos, or brand marks in generated videos.
